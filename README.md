# 🚀 Jay More - Futuristic Developer Portfolio

A sleek, modern, and highly interactive **personal portfolio website** built with **HTML, CSS, and JavaScript**. This project features a bold **violet-themed UI**, modern glassmorphism elements, and smooth dynamic animations to create a memorable user experience.

![Portfolio Preview](./preview.png) *(Add a screenshot here)*

---

## ✨ Key Highlights

* 🟣 **Violet-Centric Futuristic Design:** A bold and vibrant aesthetic tailored for a tech-focused portfolio.
* 🧊 **Glassmorphism & Neon Effects:** Premium frosted glass UI elements combined with glowing hover states.
* ⚡ **Glitch & Smooth Animations:** Engaging glitch text effects, typing animations, and scroll reveals using CSS and JavaScript.
* 🌓 **Dark / Light Mode Toggle:** Seamless theme switching for better accessibility and user preference.
* 📱 **Fully Responsive Layout:** Optimized for all screen sizes, from mobile devices to ultra-wide desktop monitors.
* 🎯 **Clean, Interactive UI:** A carefully crafted user interface prioritizing ease of navigation and readability.

---

## 📂 Sections

The portfolio is structured into the following intuitive sections:

* 🏠 **Hero:** Engaging introduction with dynamic Matrix-style canvas background.
* 👤 **About:** Detailed overview of my background, focus, and career goals.
* 🎓 **Education:** Timeline of my academic journey and qualifications.
* 🛠️ **Skills:** Categorized tech stack, tools, and domain expertise.
* 🚀 **Projects:** Showcase of my top full-stack and AI-powered applications.
* 🏆 **Achievements & Certs:** Key milestones and certifications that validate my skills.
* 💼 **Open For:** Clear indication of job preferences, roles, and preferred locations.
* 📬 **Contact:** Easy-to-use links and methods to get in touch.

---

## ⚙️ Tech Stack

This project is built using vanilla web technologies, ensuring high performance without heavy frameworks:

* **HTML5:** Semantic and accessible page structure.
* **CSS3:** Advanced styling including CSS Variables, Flexbox/Grid, animations, and responsive media queries.
* **JavaScript (ES6+):** Interactive elements, scroll observers, dynamic typing effects, and canvas rendering.

## 🚀 Run Locally

To view the portfolio on your local machine, simply clone the repository and open the HTML file:

```bash
# Clone the repository
git clone https://github.com/jaymore4501/Personal-Portfolio.git

# Navigate to the project directory
cd Personal-Portfolio

# Open index.html in your default web browser
```
