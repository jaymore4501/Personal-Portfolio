# 🚀 Futuristic Developer Portfolio

A sleek, modern **personal portfolio website** built with **HTML, CSS, and JavaScript**, featuring a bold **violet-themed UI**, glassmorphism, and smooth interactive animations.

---

## ✨ Highlights

* 🟣 Violet-centric futuristic design
* 🧊 Glassmorphism + neon hover effects
* ⚡ Glitch animations & smooth transitions
* 🌓 Dark / Light mode toggle
* 📱 Fully responsive layout
* 🎯 Clean, interactive UI experience

---

## 📂 Sections

* 🏠 Hero
* 👤 About
* 🎓 Education
* 🛠️ Skills
* 🚀 Projects
* 🏆 Achievements
* 📬 Contact
